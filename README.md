# Build With Sujan 🚀

A collection of practical web development projects built with **Next.js**.  
This repository contains beginner-friendly builds, UI components, and feature-based tutorials from my learning journey while building in public.

---

## 📌 About This Repo

This repo is tied to my YouTube channel **Build With Sujan**, where I build real-world web projects and explain them step by step.

Instead of random tutorials, each folder here represents a **small, focused project or feature** that you can learn and reuse.

---

## 🧠 Tech Stack

- Next.js
- React
- JavaScript / TypeScript
- Tailwind CSS
- Node.js (where needed)
- Supabase / Firebase (for backend features in some projects)
